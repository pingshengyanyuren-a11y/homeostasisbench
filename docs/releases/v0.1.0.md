# HomeostasisBench v0.1.0

Initial public release of HomeostasisBench.

## What this release includes

- a reproducible benchmark for multi-agent resilience under `normal`, `overload`, and `failure_storm`
- a baseline `manager -> worker -> reviewer` swarm
- a bio-inspired reference controller with `nervous`, `endocrine`, `immune`, and `metabolic` layers
- exported CSV summaries, plots, and markdown reports
- layer ablation and acceptance gates
- package install, CLI entrypoint, CI workflow, and contribution templates

## Current benchmark snapshot

- `normal`: bio completion `0.944` vs baseline `0.706`
- `overload`: bio latency `7.589` vs baseline `48.222`
- `failure_storm`: bio error `0.003` vs baseline `0.151`
- strongest layer in the current ablation run: `immune`
- weakest but net-positive layer after retuning: `endocrine`

## Suggested framing

This repository is best read as a benchmark plus reference implementation, not as a finished general-purpose agent framework.
